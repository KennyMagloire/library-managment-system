# Library Management System

## Project Overview

**Library Management System**  is a Java application whose role is to digitize and automate the operations of a library. The system enables librarians to check and manage book inventory,and track book borrowing activities.

## Features 

- ✅ Add, update, and search books
- ✅ Register and manage library members
- ✅ Borrow and return books with due date management
- ✅ Track overdue books and issue fines
- ✅ Generate reports on library usage
- ✅ File-based persistence system
